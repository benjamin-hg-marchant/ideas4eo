(Content) User Guide

dddd Input contents can be formatted using markdown and simple syntaxes. The following points present the most frequent questions to format text content:

How to add an image ?

Pour mettre une simple image dans un article il faut utiliser la balise suivante:

[image: size: caption:]

ou

  • image est l'adresse url de l'image,
  • size (optionnel) la taille entre 10 et 100% de la largeur de la page,
  • et caption (optionnel) la légende.
How to add a link ?

Pour mettre un lien dans un article il faut utiliser la syntaxe markdown suivante:

[title](url)

Lien externe

Par exemple pour un lien externe vers une page de wikipedia on peut écrire 

[article de wikipedia](https://fr.wikipedia.org/wiki/Math%C3%A9matiques)

ce qui va donner

article de wikipedia

Lien interne

Pour un lien interne la syntaxe est encore plus simple (par exemple vers l'article: Comment simplifier une fraction (Mathématiques) ?): 

[lien interne](/Articles/Comment-simplifier-une-fraction-Mathématiques/))
